Primitives

10 nodes in this area of the perspective

This hub defines the core structural primitives of the aswritten knowledge graph, centered on the [content/commentary split](https://aswritten.ai/narrative#Narrative_ContentCommentarySplit) established in July 2026. This design mandates that primary source material and authored commentary occupy distinct layers, ensuring that while the read model is already reconciled, the write practice remains in transition. Complementing this is the definition of [reconciliation as a set transaction](https://aswritten.ai/narrative#Narrative_ReconciliationAsSetTransaction), which treats commentary as a formal transaction over a specific, user-verified referent set of graph nodes. This approach prioritizes legibility, forcing the system to surface exactly what it is reasoning from. Broader architectural explorations include [shared agent memory](https://aswritten.ai/narrative#Narrative_SharedAgentMemory) for collaborative knowledge distribution and [conversational AI interview extraction](https://aswritten.ai/narrative#Observation_ApproachPattern_1) as the primary ingestion method. Several nodes, including definitions for [refract](https://aswritten.ai/narrative#Position_RefractDefinition), [scope](https://aswritten.ai/narrative#Position_ScopeDefinition), and [transaction-as-prompt](https://aswritten.ai/narrative#Position_TXIsPrompt), are registered as placeholders for future semantic grounding.
