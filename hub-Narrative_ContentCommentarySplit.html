Narrative ContentCommentarySplit

4 nodes in this area of the perspective

The hub frames a structural distinction between two layers of knowledge representation: [per-message primary source material](https://aswritten.ai/narrative#Position_ContentLayerPerMessage) transacted individually as it happens, and [authored reconciliation as commentary](https://aswritten.ai/narrative#Position_ReconciliationIsCommentaryLayer) written over a referent set after the fact. As of July 1 2026, the read model already honors this split — digests render primary-source grounding separately from a RECONCILED conclusion — but the write practice has not caught up, conflating the two by requiring memories to carry primary source content themselves. [Reconciliation as set transaction](https://aswritten.ai/narrative#Narrative_ReconciliationAsSetTransaction) develops the structural form: select a set of nodes, author something about them, surface which nodes the system is reasoning from — the legibility differentiator. By July 4, the principle had been extended outward: [the landing page IA](https://aswritten.ai/narrative#Narrative_LandingPageEpistemicArchitecture) mirrors the product's own epistemics, applying the content/commentary split to the marketing surface itself.
